这是一个人工维护的 API 速查表.

如果发现缺失某个 API, 请使用 enhancement 标签 登记.
如果发现 API 缺陷, 请使用 bug 标签 登记.

系统
GET /config/list.api →
获取服务器端配置
/sys/exception.api →
通用错误码解析/服务器存活确认
GET/POST /security/vrfy-sign.api →
测试签名
用户基础
POST /security/get-rsa-key.api →
获取 RSA 公钥
POST /user/register.api →
注册用户
POST /user/activate.api →
激活用户
POST /user/login.api →
登录/获取当前用户信息(需要密码, 稍后将分离这个接口)
POST /user/logout-via-pass.api →
使用密码登出
POST /user/logout-via-secret.api →
使用secret登出
POST /user/passwd.api →
修改密码
GET/POST /user/search.api →
搜索/查看用户(id/登录名/邮件地址...)
用户扩展
GET/POST /profile/search.api →
搜索/查看用户(id/昵称/头像)
POST /profile/update.api →
更改用户信息
藏书
POST /book/add.api →
增加藏书
POST /book/remove.api →
删除藏书
GET/POST /book/search.api →
搜索藏书
意愿
POST /desire/buy/create.api →
创建"想买"
POST /desire/sell/create.api →
创建"想卖"
GET/POST /desire/search.api →
搜索意愿
POST /desire/remove.api →
删除意愿
交易会话
GET/POST /contract/search.api →
查找交易会话
买卖

创建买卖交易会话
POST /contract/trade/accept.api →
接受买卖会话
POST /contract/trade/quit.api →
拒绝/撤销买卖会话
POST /contract/trade/deliver.api →
标记为已发货
POST /contract/trade/finish.api →
确认收货/完成交易
借阅
POST /contract/loan/create.api →
创建借阅交易会话
POST /contract/loan/accept.api →
接受借阅会话
POST /contract/loan/quit.api →
拒绝/撤销借阅会话

POST /contract/loan/deliver-ack.api →

POST /contract/loan/finish.api →
确认收货/完成交易
